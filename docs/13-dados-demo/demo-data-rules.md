# Demo Data Rules

## Regras

- Dados ficticios apenas.
- Nunca usar NIF, IBAN, email, telefone ou morada real sem consentimento.
- Passwords demo devem ser simples apenas em ambiente local/demo.
- Documentos carregados devem ser exemplos anonimizados.
- Concursos, fogos, candidatos e rendimentos devem estar alinhados com o regulamento municipal aplicavel.

## Perfis recomendados

| Perfil | Uso |
| --- | --- |
| Admin municipio | Configurar concursos, listas e backoffice. |
| Tecnico habitacao | Validar candidaturas e documentos. |
| Candidato | Submeter adesao, agregado, documentos e candidatura. |
| Inquilino | Consultar contrato, pagamentos, manutencao e vistorias. |
| Auditor | Consultar logs e evidencias sem alterar estado. |
