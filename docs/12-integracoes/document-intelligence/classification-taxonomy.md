# Document Intelligence — Taxonomia de Classificação

## Categorias suportadas

| Código | Label |
| --- | --- |
| `cartao_cidadao` | Cartão de Cidadão |
| `titulo_residencia` | Título de Residência |
| `passaporte` | Passaporte |
| `irs` | IRS |
| `nota_liquidacao` | Nota de Liquidação |
| `recibo_vencimento` | Recibo de vencimento |
| `declaracao_seguranca_social` | Declaração Segurança Social |
| `declaracao_at` | Declaração AT |
| `iban` | IBAN |
| `contrato_arrendamento` | Contrato de arrendamento |
| `comprovativo_morada` | Comprovativo de morada |
| `atestado_multiusos` | Atestado Multiusos |
| `certidao_escolar` | Certidão escolar |
| `outro` | Outro |

## Regras

- `outro` significa ausência de sinais suficientes.
- `manual_review` significa apoio à triagem, nunca rejeição automática.
- A taxonomia é administrativa e pode ser mapeada no futuro para tipos documentais municipais.
