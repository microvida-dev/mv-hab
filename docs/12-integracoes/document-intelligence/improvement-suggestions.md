# Sugestões de aperfeiçoamento documental

## Objetivo

As sugestões de aperfeiçoamento são rascunhos internos para apoiar o técnico municipal na preparação de pedidos de esclarecimento ou complemento documental.

## Estados

| Estado | Uso |
| --- | --- |
| `draft` | Sugestão gerada automaticamente e ainda não validada |
| `edited` | Sugestão editada por técnico |
| `accepted` | Sugestão aceite para eventual uso processual |
| `dismissed` | Sugestão descartada |
| `sent` | Estado reservado para futura comunicação controlada |

## Regras

- `auto_send` está desativado.
- Nenhuma sugestão é enviada ao candidato automaticamente.
- As sugestões não alteram estados da candidatura.
- O técnico pode editar, aceitar ou descartar cada sugestão.
- A auditoria regista ação, análise, score, flag e estado, sem copiar dados pessoais desnecessários.

## Linguagem permitida

Usar termos como divergência, inconsistência, necessidade de revisão manual, documento atualizado e complemento documental. Evitar linguagem acusatória.
