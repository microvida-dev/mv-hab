# Renovação Simplificada do Registo

`registration_renewals` permite ao candidato atualizar dados base sem repetir integralmente o Registo de Adesão.

## Dados editáveis

- telefone;
- telemóvel;
- tipo e validade de documento;
- morada;
- código postal;
- localidade;
- freguesia;
- município;
- nacionalidade.

## Proteções

- a renovação pertence sempre a um utilizador;
- o candidato só acede às próprias renovações;
- os dados existentes são preservados em `previous_snapshot`;
- os dados propostos ficam em `updated_snapshot`;
- a aplicação ao registo só acontece após confirmação/submissão;
- dados sensíveis não são publicados.
