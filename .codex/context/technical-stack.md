# Technical Stack

## Backend

- Laravel 13.x
- PHP 8.4
- Eloquent ORM
- Form Requests
- Policies and Gates
- Services and DTOs
- Jobs, Events and Listeners
- Queues and scheduler when needed

## Frontend

- Laravel Blade
- Tailwind CSS
- Alpine.js
- Vite

## Quality

- PHPUnit for unit and feature tests
- Pint for formatting
- PHPStan for static analysis
- Composer validation
- Route loading checks
- NPM production build
